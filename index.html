How to use:

Use the Textareas are below to specify a set of production rules for your grammar and how to transcribe the produced words. 

Production rules are specified by a string in the format of regular expressions, followed by indented strings specifying all the possible strings that can replace the string above. One of those indended strings will be picked at random, and a weight can be specified after each to skew the random picking. Is a weight not specified, it is assumed "1".

Transciption specifies how one regular expression match may be transcribed. Each match can only produce one possible transcription, so the formatting in this textarea is different. It goes "MATCH" -> "TRANSCRIPTION" each on their own line.
